# Toon_Dash

A football-themed endless runner built in Python using PyGame. You play as a Toon dodging red-card hazards and collecting gold trophies while the speed gradually increases.  

---

### Features

- Endless runner gameplay with progressive speed increase  
- Title screen, pause option and game-over screen  
- Score, lives and on-screen banner system  
- Collision detection with invincibility and screen shake  
- Particle effects for hits and trophy collection  
- Unlockable Toon Mode with new visuals and bonus life  

---

### Requirements

- Python 3.x  
- PyGame
